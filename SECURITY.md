# 安全政策

## 报告漏洞

请不要公开披露可能导致 API Key 泄漏、任意文件覆盖、目录穿越、补丁伪造或游戏文件破坏的问题。请通过仓库维护者公布的私密安全报告渠道提交复现步骤、受影响版本和影响范围。

## 安全边界

- GameTranslator 只处理用户明确选择的本地项目。
- 原游戏目录默认只读，输出写入独立工作目录。
- Provider API Key 存入操作系统凭据库。
- 游戏文本会发送到用户自行选择的模型服务；用户需要自行确认服务条款与数据政策。
- 本项目不负责解密、破解或绕过游戏资源保护。

## 不接受的报告

- 仅指出模型可能产生错误翻译。
- 针对明确标记为演示数据的统计内容。
- 要求增加资源解密或绕过保护功能。
