# 支持矩阵

## RPG Maker MV/MZ

| 能力 | 状态 | 说明 |
|---|---|---|
| 未加密部署项目识别 | 支持 | 识别 `www/data` 或 `data` |
| 地图普通对话 | 支持 | 指令码 401 |
| 对话角色名 | 支持 | 从指令码 101 获取 |
| 选项 | 支持 | 指令码 102 |
| 滚动文本 | 支持 | 指令码 405 |
| 公共事件 | 支持 | 使用相同指令白名单 |
| 物品、技能等数据库名称和说明 | 支持 | 明确字段白名单 |
| 控制码保护 | 支持 | `V`、`N`、`C`、`I` 方括号控制码 |
| 插件命令和脚本文本 | 不支持 | 不分析动态代码 |
| 加密资源 | 不支持 | 不提供解密能力 |
| 图片、字体、音频 | 不支持 | 首版只处理结构化文本 |

“支持”表示已有自动化 fixture 测试，不表示每个第三方插件或修改版引擎均兼容。扫描结果必须报告未覆盖项，不能据此宣称完整汉化。

## Ren'Py

| 能力 | 状态 | 说明 |
|---|---|---|
| 发行版目录识别 | 支持 | 识别 `renpy/`、`game/` 和游戏启动程序 |
| `.rpa/.rpyc` 剧情模板生成 | 支持 | 调用游戏自带 Ren'Py 运行时，不解包资源归档 |
| 对话和菜单字符串 | 支持 | 解析官方 `translate` 与 `old/new` 模板 |
| 插值和文本标签保护 | 支持 | 保护 `[variable]`、`{tag}` 及结束标签 |
| 标准翻译补丁 | 支持 | 导出 `game/tl/<language>/*.rpy` |
| 动态 Python 生成文本 | 不支持 | 不执行或分析任意业务脚本中的动态字符串 |
| 字体和 RTL 排版 | 不支持 | 需要针对目标游戏单独适配 |

Mayfly v0.2（Ren'Py 8.1.3、剧情位于约 1.9 GB 的 `archive.rpa`）已通过真实扫描和补丁导出验证。模板生成会短暂创建唯一命名的适配脚本，成功或失败后均清理，不写入现有语言目录。

该样本在极速模式下由旧版约 186 个串行批次降为 62 个动态批次、约 4 个并发调度波次。实际耗时仍取决于模型服务的总输出吞吐和限流策略。

## 其他引擎

Unity、Wolf RPG、Kirikiri、Unreal 和自定义引擎尚未支持。后续扩展必须实现统一引擎契约，并提供检测、提取、回写、失败路径和原文件不变测试。
