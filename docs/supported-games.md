# 支持矩阵

## RPG Maker MV/MZ

| 能力 | 状态 | 说明 |
|---|---|---|
| 未加密部署项目识别 | 支持 | 识别 `www/data` 或 `data` |
| 地图普通对话 | 支持 | 指令码 401 |
| 对话角色名 | 支持 | 从指令码 101 获取 |
| 选项 | 支持 | 指令码 102 |
| 滚动文本 | 支持 | 指令码 405 |
| 公共事件 | 支持 | 使用相同指令白名单 |
| 物品、技能等数据库名称和说明 | 支持 | 明确字段白名单 |
| 控制码保护 | 支持 | `V`、`N`、`C`、`I` 方括号控制码 |
| 插件命令和脚本文本 | 不支持 | 不分析动态代码 |
| 加密资源 | 不支持 | 不提供解密能力 |
| 图片、字体、音频 | 不支持 | 首版只处理结构化文本 |

“支持”表示已有自动化 fixture 测试，不表示每个第三方插件或修改版引擎均兼容。扫描结果必须报告未覆盖项，不能据此宣称完整汉化。

## 其他引擎

Unity、Ren'Py、Wolf RPG、Kirikiri、Unreal 和自定义引擎尚未支持。后续扩展必须实现统一 `EngineAdapter` 契约，并提供检测、提取、回写、失败路径和原文件不变测试。
